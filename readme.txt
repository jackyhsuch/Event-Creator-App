App: 
Event creator.

Features: 
1. Able to create and save event (can set event name, location, date, add friends)
2. Able to delete event
3. Edit invited friend list (delete people on view event page)

Componenets:
Navigation List - home page
Deletabe List - view event page (only if there's invited people)
Composite List - view event page
Range Button - new event page (for choosing of dates)
Back Button - almost every page
Form - new event page
Popup - when you want to delete an event/when you press invite without entering a name on new event page

Note:
1. Event list/Invite friend list are all dynamically displayed and updated
2. New event page have form components for user to fill in and information are updated dynamically in view event page
